Parágrafos, quebras de linhas, emojis e simbolos.

Para inserir um parágrafo como este, usamos a tag <p>.

Para inserir uma linha abaixo como no exemplo abaixo usamos a tag <hr>.

Para quebras o texto como no exemplo a seguir
utilizamos a tag <br>.

para inserir um emoji usamos a tag &#x + o codigo do emoji. Por exemplo, o código do emoji de caveira é 1F480, então usamos &#x + 1F480, ao juntarmos temos o emoji 💀

Para inserir simbolos usamos a tag & + nome do simbolo + ;. Por exemplo, o nome do simbolo de marca registrada é reg, então usamos & + reg + ;. Dessa maneira temos o simbolo ®

Para inserir uma imagem usamos tag <img src="imagem.png" alt="imagem/png">. Abaixo veremos um exemplo.

Para inserirmos um favicon, devemos usar a tag <link rel="shortcut icon" href="naruto.ico" type"image/x-icon"> dentro da tag <head>. O favicon fica la em cima no nome do site.
